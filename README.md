# NorbitPizza

# WPF-приложение для пиццерии

## О проекте

Это WPF-приложение, предназначенное для пиццерии, реализующее удобный интерфейс для выбора, фильтрации и заказа пиццы. Пользователь может просматривать список пицц с фотографиями и описанием, получать детальную информацию о каждом продукте, формировать заказ в корзине и оформлять покупку.

---

## Основные задачи проекта

- **Главная страница**  
  Отображение списка пицц с возможностью фильтрации по категориям, названию и ингредиентам. Каждая пицца представлена в виде блока с изображением и кратким описанием.

- **Детальный просмотр пиццы**  
  По клику на пиццу пользователь видит расширенную информацию: состав, количество калорий, вес и другую полезную информацию.

- **Корзина и оформление заказа**  
  Корзина подсчитывает итоговую сумму заказа и отображает состав выбранных продуктов. Оформление заказа реализовано либо на одной странице, либо на двух: первая — просмотр заказа и сумма, вторая — ввод адреса доставки и данных для оплаты.

## Роли и задачи в проекте

### Программисты

- Разработка пользовательского интерфейса на WPF с использованием MVVM-подхода.
- Реализация фильтрации и поиска пицц по нескольким критериям: название, категория, ингредиенты.
- Создание удобного и отзывчивого детального просмотра продукта.
- Реализация корзины с динамическим подсчётом стоимости и возможностью редактирования заказа.
- Организация навигации и логики оформления заказа.
- Обеспечение корректной работы с данными, загрузка и обновление информации о продуктах, категориях и ингредиентах.
- Интеграция с бизнес-аналитикой и тестированием — подготовка приложения для удобного сбора и обработки результатов исследований и тестов.

### Бизнес-аналитики

- Анализ финансовой окупаемости проекта.
- Исследование размещения и стратегического содержания продукта.
- Составление рекомендаций по дальнейшему развитию и масштабированию.

### Тестировщики

- Проведение UI-тестирования и ручного тестирования приложения.
